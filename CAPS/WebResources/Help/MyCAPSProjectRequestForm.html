Welcome to the Help Guide. To provide a better user experience, each section references a tab, with General being applicable to all submission categories. Tabs such as Schedule B, and PRFS are applicable to multiple submission categories, and thus have their own help section. In order for the help bubbles appear, please ensure you are on the designated tab prior to clicking the links.

Saving your changes can be done in multiple ways.

  1. Ribbon Buttons on the top left hand corner.

  2. Save button on the bottom left hand corner

  3. CTRL + S on your keyboard

  4. Auto-save, triggered every 30 seconds

Why is the name of a record just a number and a dash? The naming scheme of a project request is done in a few ways. Most will just be "Unique # - Facility" or "Unique # - Various" if the project request impacts multiple facilities. Some submission categories have different naming schemes, outlined below:

  • Site Acquisitions: "Unique # - Proposed Site"

  • New School: "Unique # - New School Name"

  • Bus: "Unique # - Transportation"
