How do I create a modular record? Are they the same as a portable?

  • Modulars are purposely built for a full-day kindergarten class which are placed permanently onto a foundation and have full bathroom facilities. To track modulars, use the "Number of Modulars" field on the desired Facility record.

The Generalized Use field will control what additional fields show and hide.

  • Class Type and Capacity fields will show if General Instruction or Capital Project Temporary Accommodation is selected

  • Portable Use Description field will show if Other Educational Instruction, Non-Instructional, or Leased is selected.
